# Models Directory

This directory is where the Whisper models will be automatically downloaded when you first run the application.

Models are downloaded by `faster-whisper` on first use and cached here for future runs.

Typical model sizes:
- tiny.en: ~75 MB
- base.en: ~145 MB
- small.en: ~470 MB
- medium.en: ~1.5 GB

The models are automatically downloaded from Hugging Face and do not need to be manually placed here.
